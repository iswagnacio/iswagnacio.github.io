Project 1

Images of the Russian Empire: Colorizing the Prokudin-Gorskii photo collection

Project Overview

This project implements an automatic image alignment system to reconstruct color photographs from Sergei Mikhailovich Prokudin-Gorskii's pioneering collection of early 20th century Russian Empire photographs. Each glass plate negative contains three separate exposures of the same scene taken through red, green, and blue filters.

Methodology

1. Image Preprocessing

  1. Channel Extraction: Divide the input image into three equal vertical sections (B, G, R from top to bottom)
  2. Border Cropping: Remove the outer 10% of each image to eliminate irregular borders that could interfere with alignment
  3. Edge Detection: Apply Sobel operators to extract structural features

2. Sobel Edge Detection (bells and whistles)

The Sobel operator detects edges by computing the gradient of image intensity at each pixel using convolution with two 3×3 kernels. Without edge detection, the simple NCC pixel alignment works well on jpg files but fails on the high-res tif files.

Sobel Operators

The Sobel operators are defined as two 3×3 convolution kernels:

Horizontal Edge Detection (Gx)

⎡ -1 0 1 ⎤ ⎢ -2 0 2 ⎥ ⎣ -1 0 1 ⎦

Vertical Edge Detection (Gy)

⎡ -1 -2 -1 ⎤ ⎢ 0 0 0 ⎥ ⎣ 1 2 1 ⎦

Mathematical Formulation

For an image I(x,y), the Sobel edge detection process computes:

  1. Horizontal Gradient: Gx = Sx * I(x,y)
  2. Vertical Gradient: Gy = Sy * I(x,y)
  3. Edge Magnitude: |G| = √(Gx² + Gy²)

Where * denotes the convolution operation.

3. Normalized Cross-Correlation (NCC)

For alignment scoring, we use NCC which measures the correlation between edge maps:

NCC Formula: NCC(A,B) = Σ(A-Ā)(B-B̄) / (||A-Ā|| × ||B-B̄||)

4. Multi-Scale Pyramid Alignment

For high-resolution images, exhaustive search becomes computationally prohibitive. The pyramid approach provides an efficient solution:

  1. Pyramid Construction: Create multiple resolution levels by iteratively downsampling by factor of 2
  2. Coarse Alignment: Start with exhaustive search on the lowest resolution (≈25px minimum dimension)
  3. Refinement: For each higher resolution level:
    • Scale up the displacement from previous level (×2)
    • Search in a small 3×3 window around the scaled position
    • Refine the alignment incrementally
